Lenguajes de Programación

A continuación, se describe cómo manejan las cadenas en diferentes lenguajes de programación:

Python

cadena = "Hola Mundo"
print(cadena.upper()) 
 # Convertir a mayúsculas

Java

String cadena = "Hola Mundo";
System.out.println(cadena.toUpperCase());  
// Convertir a mayúsculas

C++

#include <iostream>
#include <string>
using namespace std;
int main() {
    string cadena = "Hola Mundo";
    for (char &c : cadena) c = toupper(c);
    cout << cadena << endl; 
     // Convertir a mayúsculas
    return 0;
}

JavaScript

let cadena = "Hola Mundo";
console.log(cadena.toUpperCase()); 
 // Convertir a mayúsculas

PHP

$cadena = "Hola Mundo";
echo strtoupper($cadena); 
 // Convertir a mayúsculas

Ruby

cadena = "Hola Mundo"
puts cadena.upcase  # Convertir a mayúsculas

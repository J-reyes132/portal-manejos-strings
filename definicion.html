Definición de Cadenas

Una cadena (string) es una secuencia de caracteres utilizada para representar texto en programación. Las cadenas son inmutables, lo que significa que no pueden modificarse después de su creación.

Características Principales

  • Inmutabilidad: Las cadenas no pueden modificarse después de su creación.
  • Indexación: Se puede acceder a cada carácter mediante un índice.
  • Concatenación: Unir dos cadenas para formar una nueva.
  • Longitud: Se puede obtener la cantidad de caracteres en una cadena.

Ejemplos de Cadenas

Python

cadena = "Hola Mundo"
print(cadena)  # Salida: Hola Mundo

JavaScript

let cadena = "Hola Mundo";
console.log(cadena);  // Salida: Hola Mundo

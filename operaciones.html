Operaciones de Cadenas

Las operaciones más comunes con cadenas son:

Operaciones Básicas

  • Concatenación: Unir dos cadenas.
  • Búsqueda: Encontrar una subcadena dentro de una cadena.
  • Sustitución: Reemplazar una subcadena por otra.
  • División: Dividir una cadena en partes.

Ejemplos de Operaciones

Concatenación en Python

cadena1 = "Hola"
cadena2 = "Mundo"
print(cadena1 + " " + cadena2)  # Salida: Hola Mundo

Búsqueda en JavaScript

let cadena = "Hola Mundo";
console.log(cadena.indexOf("Mundo"));  // Salida: 5

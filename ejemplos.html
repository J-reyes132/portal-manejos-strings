Ejemplos de Código

A continuación, se presentan dos ejemplos de programas que implementan algoritmos de manejo de cadenas:

Invertir una Cadena 

cadena = "Hola Mundo"
print(cadena[::-1])  # Salida: "odnuM aloH"

Contar Vocales en una Cadena 

cadena = "Hola Mundo"
vocales = sum(1 for c in cadena if c.lower() in 'aeiou')
print(vocales)  # Salida: 4
